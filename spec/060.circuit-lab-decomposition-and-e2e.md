# SPEC 060: Circuit Lab Decomposition and E2E Regression

## Status
Implemented.

## Scope
This specification reintroduces circuit decomposition and adds regression tests for the most important user flows.

## Implementation
- `circuitDecomposition.ts` decomposes SWAP and controlled phase gates, preserves parameterized rotations, merges adjacent rotations, and cancels adjacent self-inverse gates.
- Circuit Lab exposes a decomposition and optimization report in the Source Code inspector.
- `frontend/playwright.config.ts` and `frontend/e2e/qav-critical-flows.spec.ts` define critical route, Formula Studio, Circuit Lab, and Hardware flows.
- `package.json` includes `test:e2e` and `@playwright/test`.

## Verification
Static audit confirms the decomposition module, Playwright config, and E2E specification are present.
