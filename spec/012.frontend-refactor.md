# SPEC 012: Frontend Refactor - Eliminasi God Files & God Functions

## 1. Task

Refactor frontend codebase untuk menghilangkan "god files" dan "god functions". File page yang terlalu besar dipecah menjadi komponen-komponen kecil yang lebih maintainable.

## 2. Goals

1. **Eliminate God Pages:** Pecah CombinedPage (500-880 lines) menjadi file-file kecil
2. **Extract Chart Components:** Pindahkan visualisasi inline ke folder `components/charts/`
3. **Create Algorithm Hooks:** Buat custom hooks untuk setiap algoritma (QAOA, VQE, QFT)
4. **Shared Layout Components:** Buat komponen layout yang reusable
5. **Maintain Functionality:** Semua fitur existing tetap berjalan setelah refactor

## 3. Why

### Masalah

- `QAOACombinedPage.tsx` = 880 lines, terlalu besar
- `QFTCombinedPage.tsx` = 549 lines
- `VQECombinedPage.tsx` = 543 lines
- Chart/visualization components masih inline di dalam page, jadi tidak reusable
- Belum ada hooks untuk QAOA, VQE, dan QFT, hanya DJ yang sudah ada
- Ada duplikasi logika `sortCaseIds` dan download handler di setiap page

### Dampak

- Sulit maintenance dan debugging
- Komponen chart tidak bisa direuse untuk algoritma lain
- Business logic tercampur dengan presentation logic
- Code review jadi sulit karena file terlalu panjang

```typescript
// QAOACombinedPage.tsx after refactor (~100 lines)
export default function QAOACombinedPage() {
  const {
    selectedCaseId,
    availableCases,
    benchmarkResult,
    isLoading,
    error,
    activeTab,
    setActiveTab,
    setSelectedCaseId,
    handleRun,
    handleDownload
  } = useQAOA();

  return (
    <AlgorithmPageLayout>
      <AlgorithmHeader
        title="Quantum Approximate Optimization Algorithm"
        icon={GitBranch}
        selectedCaseId={selectedCaseId}
        availableCases={availableCases}
        onCaseChange={setSelectedCaseId}
        onRun={handleRun}
        onDownload={handleDownload}
        isLoading={isLoading}
        hasResult={!!benchmarkResult}
      />

      <ClassicQuantumTabs activeTab={activeTab} onChange={setActiveTab} />

      {activeTab === 'classic' && (
        <QAOAClassicTab result={benchmarkResult} />
      )}

      {activeTab === 'quantum' && (
        <QAOAQuantumTab result={benchmarkResult} />
      )}
    </AlgorithmPageLayout>
  );
}
```
