### Backend (COMPLETE)

**1. `backend/services/qft_service.py`**
- Added `_sv_to_dict_list()` helper function for JSON serialization of complex statevector amplitudes
- Added `_extract_qubit_phases()` function to extract accumulated phase angles per qubit from statevector
- Added `build_qft_animation_timeline()` function - generates step-by-step timeline with:
  - Phase 1: Signal initialization (load amplitudes to statevector)
  - Phase 2: Phase cascade (controlled-phase rotations, n(n-1)/2 steps)
  - Phase 3: Hadamard layer (n steps)
  - Phase 4: SWAP network (n/2 steps for bit reversal)
  - Phase 5: Measurement (1 step)
- Added `_build_qft_animation_partitions()` function - groups timeline steps by phase
- Added `get_qft_animation_payload()` function - builds complete animation payload with timeline, partitions, FFT comparison, and measurement data

## Key Differences from DJ Animation

| Aspect | DJ | QFT |
|--------|-----|-----|
| Input | Truth table (f(x) → {0,1}) | Signal vector (amplitudes) |
| State tracking | Wire markers (0/1/+) | Phase angles (φ) per qubit |
| Key visualization | Oracle box, MCX controls | Phase wheels, angle accumulation |
| Output | Constant/Balanced classification | Frequency spectrum |
| 3D Scene | Bloch spheres on wires | Signal bars + qubit orbs + phase bands |
