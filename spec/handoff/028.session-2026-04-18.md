## Detail Implementasi

### A. Helper derivasi dinamis

`buildOracleConstructionModel(data)` dibuat untuk menghasilkan model penjelasan dari payload animasi yang sudah ada.

Mode yang ditangani:

1. `constant-zero`
   - oracle = identity
2. `constant-one`
   - oracle = satu X pada ancilla
3. `balanced`
   - satu row `f(x)=1` menjadi satu blok oracle

### B. Panel UI baru

`OracleConstructionPanel` menampilkan:

1. intro ringkas,
2. langkah bernomor,
3. ringkasan hasil konstruksi,
4. daftar blok oracle per input bernilai `1`.

### C. Sinkron dengan animasi

Panel menerima `activeBits` dari langkah aktif animasi, lalu memberi highlight pada pola input yang sedang diproses.

## Kenapa Solusi Ini Aman

1. Tidak mengubah kontrak backend.
2. Tidak mengubah logika simulasi atau builder circuit.
3. Hanya menambahkan lapisan penjelasan di frontend berdasarkan data yang sudah ada.
4. Jika dataset baru ditambahkan atau truth table diubah, panel ikut berubah otomatis karena derivasi dilakukan saat render.
