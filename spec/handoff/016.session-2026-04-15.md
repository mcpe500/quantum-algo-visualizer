## Summary

Alignment dan susunan visual halaman `/dj/topography` diperbaiki tanpa mengubah tema warna. Fokus utama: ritme kartu, baseline matriks, dan grouping konten berdasarkan level qubit.

---

## What Changed

### 1) Matrix body alignment

File: `frontend/src/components/dj/OracleTopography.tsx`

- `items-center` → `items-start`
- Grid matrix ditambah `content-start`

Efek: kartu 3-qubit tidak lagi terlihat "melayang" di tengah.

### 2) Layout restructure by qubit group

File: `frontend/src/components/dj/OracleTopography.tsx`

- Cases dikelompokkan berdasarkan `n_qubits`
- Render jadi 2 section:
  - 3 Qubit Oracle Cases
  - 4 Qubit Oracle Cases
- Tiap section pakai grid 2 kolom (`xl:grid-cols-2`)

Efek: kepadatan 8-state dan 16-state tidak lagi bentrok dalam satu baris.

### 3) Numeric alignment polishing

File: `frontend/src/components/dj/OracleTopography.tsx`

- Tambah `tabular-nums` pada teks input dan output sel.

Efek: digit terlihat lebih stabil antar sel dan antar baris.

### 4) Existing ordering fix retained

- Truth table tetap sorted by binary value ascending.
