### 2. Standardized Spacing Tokens

**`constants.ts` exports:**

```typescript
// Spacing
VQE_SPACING.sectionGap    // 'mb-6'
VQE_SPACING.cardGap       // 'gap-4'
VQE_SPACING.innerPadding  // 'p-6'
VQE_SPACING.compactPadding // 'p-4'

// Grid configs
VQE_GRID.metrics      // 'grid-cols-2 md:grid-cols-4'
VQE_GRID.comparison   // 'grid-cols-1 md:grid-cols-2'
VQE_GRID.energy       // 'grid-cols-1 md:grid-cols-3'

// Typography (replacing hardcoded text-[10px], text-[11px])
VQE_TYPOGRAPHY.tiny     // 'text-[10px] uppercase...'
VQE_TYPOGRAPHY.small    // 'text-xs text-slate-500'
VQE_TYPOGRAPHY.caption  // 'text-xs text-slate-400'

// Dimensions (replacing magic numbers)
VQE_DIMENSIONS.timelineMinWidth           // 80 (was min-w-[80px])
VQE_DIMENSIONS.timelineMinWidthMd         // 100 (was min-w-[100px])
VQE_DIMENSIONS.stepFlowMinHeight          // 280 (was minHeight: 280)
VQE_DIMENSIONS.stepFlowNodeWidth          // 110 (was w-[110px])
VQE_DIMENSIONS.stepFlowConnectorPadding   // 105 (was pr/pl-[105px])

// Colors (semantic variants)
VQE_COLORS.success  // 'text-emerald-600 bg-emerald-50...'
VQE_COLORS.warning  // 'text-amber-600 bg-amber-50...'
VQE_COLORS.info     // 'text-blue-600 bg-blue-50...'
VQE_COLORS.error    // 'text-red-600 bg-red-50...'
```

## Design Decisions

1. **Layout primitives vs inline divs**: Component-based approach memungkinkan konsistensi spacing/grid di seluruh VQE pages
2. **Token constants**: Centralized di `constants.ts`, memudahkan future design system updates
3. **Variant-based cards**: `VQECard` dengan variant (success/warning/info/error) menggantikan inline conditional classes
4. **Max-width 1400px**: Align dengan FormulaStudioPage untuk konsistensi layout
