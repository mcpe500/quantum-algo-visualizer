## Perubahan Utama

### `backend/services/vqe_service.py`

- Rebuild benchmark payload ke shape frontend:
  - `classical`, `quantum`, `comparison`
- Reintroduce exact classical baseline (FCI diagonalization)
- Reintroduce variational optimization + convergence history
- Reintroduce trace payload (`stages`, `partitions`)
- Circuit image payload disesuaikan kembali (`image`, `depth`, `gate_count`, ansatz metadata)

### `backend/services/qft_service.py`

- Rebuild benchmark payload ke shape frontend:
  - `fft`, `qft`, `comparison`, plus top-level signal metadata
- Qubit size dihitung dari `log2(n_points_padded)`, bukan jumlah titik mentah sebagai jumlah qubit
- Tambah trace payload (`stages`, `partitions`) yang konsisten
- Circuit image payload kembali menyertakan metadata QFT yang dibutuhkan frontend

### `backend/services/qaoa_service.py`

- Tambah normalisasi edge untuk format:
  - `{u,v,weight}`
  - `[u,v]`
  - `[u,v,weight]`
- Semua pipeline memakai problem normalized yang sama:
  - benchmark
  - trace
  - circuit image
  - classical run
- Rebuild benchmark payload (`exact/classical/quantum/comparison`) sesuai frontend
- Trace payload kembali ke shape `n_nodes/n_edges/p_layers/stages/partitions`
