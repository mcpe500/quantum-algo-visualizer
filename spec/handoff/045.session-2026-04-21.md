## What Was Done

### Shared Gate Layer
- Ubah representasi gate shared yang sebelumnya non-box menjadi box:
  - `LabeledSphereGate` -> box geometry
  - `LabeledDiscGate` -> box geometry
  - `ControlDot` -> box berlabel `C`
  - `TargetMarker` -> box berlabel `X`
- Label gate selalu berada di dalam box.
- Ukuran gate dapat dikontrol seragam lewat `size`.

### DJ Scene
- Marker controlled gate `●` dan `⊕` sekarang dirender sebagai box (bukan titik/cincin).
- Koneksi vertikal controlled gate tetap dipertahankan.

### QFT Scene
- Gate QFT diganti ke boxed render penuh:
  - `ENC` (init)
  - `H`
  - `C` + `CP` untuk controlled-phase
  - `SWAP` + koneksi vertikal
  - `M` (measurement)
- `gateSize` dihitung dari gap kolom agar konsisten antargate.

### QAOA Scene
- Gate `C`, `ZZ`, `RX`, `M` diubah ke boxed render.
- Superposition `H` memakai ukuran gate yang sama.
- Cost edge tetap terhubung vertikal antar qubit terlibat.
