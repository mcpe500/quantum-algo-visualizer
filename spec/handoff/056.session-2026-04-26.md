## WHAT WAS DONE

### Phase A: Critical + Layout (5 fixes) ✅

**Fix 1: crypto.randomUUID() fallback** — `useCanvasReducer.ts:4-9`
- Added `typeof crypto.randomUUID === 'function'` guard
- Fallback: `'id-' + Date.now().toString(36) + '-' + Math.random().toString(36).slice(2, 11)`
- Prevents crashes on HTTP/browser-old environments

**Fix 2: dropPointRef race condition** — `StudioCanvas.tsx`
- Removed `dropPointRef` entirely (was written in `handleCanvasDrop`, read in `handleDragEnd` — classic race)
- `handleCanvasDrop` is now a no-op
- `handleDragEnd` uses only `active.rect.current.translated` from dnd-kit (authoritative source)

**Fix 3: FormulaInspector w-72 → w-96** — `NodeInspector.tsx:411`
- `w-72` → `w-96 shrink-0`
- Now consistent with InputInspector and ExpressionInspector

**Fix 4: DroppableCanvas min-w-0** — `StudioCanvas.tsx:58`
- Added `min-w-0` to canvas flex child
- Prevents canvas from refusing to shrink in narrow viewports

**Fix 5: Inspector wrapper h-full** — `StudioCanvas.tsx:495`
- Added `h-full` to `<div className="shrink-0 h-full">`
- Ensures inspector panel stretches to full parent height
