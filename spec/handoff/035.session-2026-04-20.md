# Handoff 035: Circuit Lab Export Preview and JSON Save

**Date:** 2026-04-20
**Status:** COMPLETE

## Summary

Circuit Lab kini mendukung kontrol jumlah qubit dengan tombol `+/-`, penyimpanan project sebagai JSON, dan preview ekspor syntax untuk `Qiskit`, `Cirq`, serta `ProjectQ`. Output ekspor tidak lagi tersembunyi sebagai file unduhan saja, tetapi tampil langsung di UI melalui panel `Export Preview`.

## What Was Done

### 1. Added serializer and code exporter in builder hook

**`frontend/src/components/qubit-playground/useCircuitBuilder.ts`**

- Added `CircuitProjectData`
- Added `CircuitCodeExportFormat`
- Added `sortPlacements(...)` for deterministic export order
- Added `exportProjectData()`
- Added `exportCircuitCode(format)`
- Added framework exporters for:
  - JSON
  - Qiskit
  - Cirq
  - ProjectQ

### 2. Added qubit increment/decrement API

**`frontend/src/components/qubit-playground/useCircuitBuilder.ts`**

- Added:
  - `canIncreaseQubits`
  - `canDecreaseQubits`
  - `increaseQubits()`
  - `decreaseQubits()`

This replaces the need for fixed `1/2/3` selector buttons in Circuit Lab UI.

### 3. Updated Circuit Lab toolbar and export UI

**`frontend/src/components/qubit-playground/CircuitLabTab.tsx`**

- Replaced qubit selector buttons with `+/-` control
- Added quick action `Save JSON`
- Added `Export Preview` panel with tabs:
  - `JSON`
  - `Qiskit`
  - `Cirq`
  - `ProjectQ`
- Added download button for currently active export format
- Added monospaced code preview that updates live with circuit changes

### 4. Reused existing download utility

**`frontend/src/utils/download.ts`**

- No code change needed
- Reused existing `downloadBlob(...)` utility for `.json` and `.py` output

## Notes

1. Export preview is live and derived from the current circuit model.
2. JSON save is one-way download only; import-from-JSON is not implemented in this session.
3. Qubit range remains `1..3`, consistent with current builder and simulator scope.
4. ProjectQ controlled phase is exported using `Control` + `R(theta)` on target qubit.
