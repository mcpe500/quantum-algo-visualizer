## Detail Implementasi

### A. Enhanced OracleConstructionModel

Model baru punya struktur lebih kaya:

- `profile`: 'constant-zero' | 'constant-one' | 'balanced'
- `overviewBlock`: { label, content, gateCount } — ringkasan cepat di atas
- `rows[]`: tiap row punya:
  - `stepNumber`: nomor urut blok
  - `intro`: penjelasan satu baris tentang pola ini
  - `subSteps[]`: array 4 fase (init/flip/mcx/restore) dengan detail
  - `gateSequence`: notation seperti `[X q0,q1] → [MCX] → [X q0,q1]`
  - `zeroControls[]`: list qubit yang perlu flip sementara

### B. Panel Styling Updates

Semua panel styling di-update untuk match redesign concept:

- `border-gray-200` instead of `border-slate-200`
- Text 10-12px labels dengan tracking-widest uppercase
- Compact padding (p-3, p-4 instead of px-4 py-4)
- Table-style layouts untuk TruthTablePanel dan StateSummaryPanel
- bg-gray-50 header sections

### C. Panel Consolidation

StateSummaryPanel dihapus karena info sudah tercakup dalam ReadingGuideCard + MeasurementPanel.

FinalAmplitudePanel di-merge ke MeasurementPanel — top 4 input states ditampilkan dalam measurement panel dengan progress bars.

Right column sekarang fixed 380px dengan 4 stacked panels.
