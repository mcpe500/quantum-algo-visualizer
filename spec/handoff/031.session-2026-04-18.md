## Dataset and Proposal Alignment

Sudah diselaraskan dengan `proposal.md`:

1. QAOA diperlakukan sebagai algoritma hybrid, bukan alur linear.
2. Shots ditampilkan sebagai `1024`.
3. Studi kasus tetap Max-Cut graph 3 dan 4 node.
4. Mapping ke Hamiltonian Ising ditampilkan eksplisit.
5. Comparator utama tetap `Simulated Annealing`.
6. Brute force ditampilkan sebagai optimum referensi.

## Notes

1. Implementasi animasi saat ini sudah generik untuk `p_layers > 1`, meskipun dataset sekarang masih `p=1`.
2. Visual animasi memakai hasil optimizer nyata, tetapi tetap diringkas ke checkpoint agar user non-teknis mudah mengikuti loop hybrid.
3. Struktur kontrak dibuat cukup netral agar nanti bisa diperluas ke jalur multi-framework jika backend QAOA lintas framework ditambahkan.
