# Handoff 023: DJ MP4 Export via FFmpeg.wasm

Date: 2026-04-17
Session: Tambah export MP4 menggunakan FFmpeg.wasm + perbaikan Bloch sphere camera mode

## Goal

Menambahkan opsi export video MP4 untuk animasi Deutsch-Jozsa, selain WebM yang sudah ada. Juga memperbaiki camera mode pada preview canvas.

## Ringkasan Hasil

1. Dua tombol export: "WebM 1080p" (instant) dan "MP4 1080p" (rekam + konversi)
2. FFmpeg.wasm di-load lazy dari CDN (~25MB pertama kali, di-cache setelahnya)
3. Vite config ditambah COOP/COEP headers untuk SharedArrayBuffer
4. Preview canvas sekarang benar menggunakan `cameraMode` user (tidak locked fixed)
5. Export canvas tetap locked fixed (predictable framing)

## Keputusan Penting

### 1. FFmpeg.wasm client-side, bukan backend

Dipilih karena self-contained, tidak perlu ubah backend, dan cocok untuk proyek TA. Tradeoff: ~25MB WASM download pertama kali.

### 2. Dua tombol, bukan satu

User minta dua tombol terpisah. WebM instant, MP4 perlu konversi. Tidak ada fallback ke WebM jika MP4 gagal.

### 3. Preview camera fix

Preview canvas sebelumnya terkunci ke `fixed` camera. Diperbaiki agar menggunakan `cameraMode` state user — orbit untuk eksplorasi manual, fixed untuk auto-play.
