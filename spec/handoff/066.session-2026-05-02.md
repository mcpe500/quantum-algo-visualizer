# Handoff 066: VQE-FCI Computational Trace

## Summary

Implemented VQE-FCI computational trace based on the step-by-step structure of the provided source document. Existing VQE ansatz gate trace remains; new trace explains the full computation path: molecule input -> FCI exact diagonalization -> VQE variational loop -> shot-based evaluation -> FCI/VQE comparison.

## User Request

User asked to implement tracing in `quantum-algo-visualizer` like `llm-wiki-agent/wiki/sources/vqe-fci-computational-trace.md`, following `spec/prompts/INSTRUCTIONS.md`.

## Important Decision

The source document contains static numeric values that differ from the current canonical VQE runtime data. Implementation follows the source trace structure, but uses active runtime values from the project:

- VQE-01 FCI runtime: about `-1.915284 Ha`
- VQE-02 FCI runtime: about `-0.539423 Ha`

This avoids stale or conflicting numbers in UI.

## Backend Changes

### `backend/services/vqe_service.py`

Added computational trace helpers:

- `_format_number`
- `_trace_value`
- `_matrix_preview`
- `_pauli_terms_payload`
- `_iteration_trace_payload`
- `_step`
- `_build_vqe_fci_computational_trace`

`run_vqe_payload()` now:

1. Loads raw VQE case in addition to canonical case.
2. Runs existing FCI exact diagonalization.
3. Runs existing VQE optimization.
4. Runs existing shot-based evaluation if available.
5. Builds `computational_trace` from runtime data.
6. Adds `computational_trace` to benchmark response.

Added:

```python
def get_vqe_computational_trace_payload(case_id, shots=1024):
    payload = run_vqe_payload(case_id, shots)
    if not payload:
        return None
    return payload.get('computational_trace')
```

### `backend/api/routes/vqe.py`

Added route:

```text
GET /api/vqe/computational-trace/<case_id>?shots=1024
```

This route runs the same pipeline and returns only the computational trace object.

### FCI Steps

1. Membaca Molekul dan Basis Set
2. Membangun Orbital Molekul
3. Menyusun Determinan Slater
4. Membangun Hamiltonian Qubit
5. Diagonalisasi Matriks Hamiltonian
6. Menetapkan Referensi Klasik

### VQE Steps

1. Parsing Konfigurasi VQE
2. Mapping Hamiltonian ke Pauli Sum
3. Membangun Ansatz Parameterized Circuit
4. Evaluasi Expectation Value
5. Optimasi Parameter COBYLA
6. Evaluasi Shot-Based
7. Membandingkan VQE terhadap FCI
