## Diagnosis Notes

Why `tsc` and `vite build` worked but dev server failed:
1. **tsc** — TypeScript compiler has explicit extension probing: tries `.ts`, `.tsx`, `.d.ts` in sequence
2. **vite build (Rolldown)** — builds full dependency graph before resolving; bundler's full-graph analysis probes more extensions than dev server
3. **vite dev server** — per-file on-demand resolution; when specifier ends in `.types`, Rolldown resolver treats it as complete extension and never probes `.ts`
