## Key Architecture Decisions

1. **Circuit-diagram over zone panels**: Physical circuit layout is more intuitive for understanding quantum computing. Gates at fixed positions = familiar circuit diagram metaphor.

2. **computeOrbX() interpolation**: Within each phase, orb X position interpolates linearly between phase start and end X. This gives smooth left-to-right movement without continuous progress variable.

3. **Real Qiskit data preserved**: All probability data from backend statevector snapshots. No hardcoded states.

4. **Dataset from JSON**: Data comes from `datasets/dj/DJ-*.json` via backend API. No hardcoded datasets.

5. **Superposition visual**: wireframe + reduced opacity + glow + bobbing when qubit is in superposition (0 < p1 < 1). From reference code concept.
