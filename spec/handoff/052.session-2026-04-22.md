## Design Decisions

1. **Boundary dataset vs benchmark**: Dataset page (`/vqe/dataset`) hanya menampilkan raw case, preprocessing transform, dan struktur Hamiltonian. Hasil FCI/VQE runtime tetap di halaman benchmark (`/vqe`).
2. **Canonical schema dataset-only**: Tidak menyertakan `classical_reference`/`quantum_reference` di canonical JSON. Alasan: canonical adalah input data, bukan hasil eksperimen.
3. **FCI correctness**: FCI dihitung runtime oleh `vqe_service.py` via exact diagonalization (`eigvalsh`) dari Hamiltonian yang sama dipakai VQE. Perbandingan matematis valid.
4. **Pauli term visualization**: Operator badges dengan color-coding (I=slate, Z=blue, X=red, Y=green) untuk membantu pembaca memahami struktur Hamiltonian.
5. **Consistency dengan algoritma lain**: Dataset page VQE mengikuti pola yang sama dengan DJ (truth table), QFT (waveform), QAOA (adjacency matrix) — semuanya explanatory, bukan runtime results.
