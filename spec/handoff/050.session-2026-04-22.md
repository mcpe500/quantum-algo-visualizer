## Key Design Decisions

1. **adjacency_matrix sebagai required field di JSON**: Data harus tersedia langsung
2. **Fallback di backend**: Jika matrix tidak ada di JSON, backend compute dari edges
3. **Optional prop di frontend**: `adjacencyMatrix?` - backward compatible
4. **3 interfaces diupdate**: QAOAGraph, QAOABenchmarkResult, QAOAAnimationPayload

## Data Flow (After Changes)

  ↓
