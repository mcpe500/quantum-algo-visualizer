## What Was Done

### 1. QFTFlowDiagram Component (Baru)
- File: `frontend/src/components/qft/QFTFlowDiagram.tsx`
- Menampilkan 4 step transformasi FFT secara visual:
  - Sinyal Input (domain waktu)
  - Zero-Padding (persiapan 2^n)
  - FFT Algorithm (O(N log N))
  - Frekuensi Dominan (hasil analisis)
- Setiap step dihubungkan panah dengan label proses
- Responsive: horizontal di desktop, vertical di mobile
- Warna gradient dari biru ke amber menunjukkan progress transformasi

### 2. DominantBinsExplanation Component (Baru)
- File: `frontend/src/components/qft/DominantBinsExplanation.tsx`
- Komponen interaktif dengan 3 section accordion:
  - **"Apa itu Frequency Bin?"** - Penjelasan konsep bin sebagai slot frekuensi
  - **"Kenapa 'Dominant'?"** - Penjelasan magnitude dan analogi nada paling keras
  - **"Kenapa Ada 2 Bin Sama?"** - Penjelasan Conjugate Symmetry (bin k ↔ bin N-k)
- Analysis panel menampilkan grid bin cards dengan:
  - Nomor bin
  - Magnitude
  - Frekuensi terhitung (f = bin/N)
  - Marker "(mirror)" untuk pasangan symmetric

### 3. SpectrumChart Enhancement
- File: `frontend/src/components/charts/SpectrumChart.tsx`
- Menambahkan prop `highlightBins: number[]`
- Dominant bins ditampilkan dengan:
  - Warna amber (#f59e0b) vs blue (#2563EB) untuk bins lainnya
  - Opacity penuh (1.0) vs 0.8
  - Label angka di atas bar
- Legend ditampilkan di bawah chart untuk membedakan "Dominant Bin" vs "Other Bins"

### 4. Visual Connector Antara Charts
- Panah besar (ArrowRight) di tengah antara Input Signal dan FFT Spectrum
- Background bulat putih dengan border biru
- Hanya muncul di desktop (hidden di mobile dengan `hidden md:block`)

### 5. QFTClassicTab Integration
- File: `frontend/src/components/qft/QFTClassicTab.tsx`

  - Flow diagram di atas metrics

  - Visual connector antar chart

## Design Philosophy
Visualisasi ini dirancang dengan prinsip "educational first":
- Setiap elemen memiliki tujuan pembelajaran yang jelas
- Flow diagram membantu pemahaman alur transformasi
- Penjelasan interaktif mengurangi cognitive load
- Highlight visual memudahkan identifikasi hasil penting
