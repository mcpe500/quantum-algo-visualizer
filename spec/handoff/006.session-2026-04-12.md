## Summary

Added a grouped quantum preview below the existing DJ quantum solution. The new preview is live, boxed, and derived from the active trace data, so it updates with the selected JSON case.

## What Was Done

### 1. New grouped preview component

- Created `frontend/src/components/dj/QuantumPhaseGroupedPreview.tsx`
- Renders phase cards for `init / prep / oracle / interference / measure`
- Slices trace stages using `trace.partitions`
- Shows step numbers, operations, input qubits, and ancilla state

### 2. Page integration

- Updated `frontend/src/pages/DJCombinedPage.tsx`
- The existing quantum solution remains unchanged
- The new grouped preview now renders directly below it
- The preview is inside the capture area, so PNG export includes it

### 3. Dynamic behavior

- Uses the current DJ trace returned by the backend
- Adapts to the active JSON case
- Works for future cases like `DJ-05`, `DJ-06`, `DJ-10`, etc.
