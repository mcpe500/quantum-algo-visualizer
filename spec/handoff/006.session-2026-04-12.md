# HANDOFF 006: DJ Phase-Boxed Circuit Image

**Date:** 2026-04-12
**Status:** COMPLETE ✅

---

## Summary

Implemented phase-boxed circuit images for DJ algorithm. Phase boxes (init, prep, oracle, interference, measure) are now drawn directly on the matplotlib circuit image in the backend, not as a separate frontend component.

## What Was Done

### 1. Backend phase box drawing

- Added `_add_phase_boxes_to_figure(fig, qc)` helper function
- Function analyzes circuit data to find barrier positions
- Splits circuit into 5 phases dynamically based on gate structure
- Draws colored Rectangle patches for each phase
- Adds numbered phase labels at top of each box
- Handles empty oracle case (constant-0) with minimum visual weight

### 2. New boxed circuit endpoint

- Added `/api/dj/circuit-image-boxed/<case_id>` endpoint
- Returns circuit PNG with phase boxes baked in
- Same response format as regular circuit-image endpoint

### 3. Frontend integration

- Added `getCircuitImageBoxed()` method to API service
- Updated `DJCombinedPage.tsx` to load and display boxed circuit
- Boxed image appears below regular quantum solution
- Both images included in PNG download (inside capture-area)

### 4. Removed wrong implementation

- Deleted `QuantumPhaseGroupedPreview.tsx` (wrong approach)
- Removed wrong component usage from page

## Dynamic Behavior

The implementation is fully dynamic:

- Phase boxes derived from actual circuit gate structure
- Box widths proportional to instruction count per phase
- Automatically adapts to any DJ case (DJ-01 through DJ-10+)
- Oracle phase shows even if empty (constant-0)
- Colors: blue/init, green/prep, amber/oracle, violet/interference, red/measure
