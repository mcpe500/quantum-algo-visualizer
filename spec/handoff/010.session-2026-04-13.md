## Answer: QFT vs FFT with Same Dataset — Are Results Different?

**Yes, but in expected and explainable ways:**

1. **FFT output is deterministic** — produces exact complex-valued spectrum every run
2. **QFT output is probabilistic** — produces measurement counts that approximate the Fourier spectrum
3. **QFT uses amplitude encoding** — the signal values become quantum state amplitudes (normalized), not raw values
4. **QFT result is a probability distribution** — you see which basis states are measured most, which corresponds to dominant frequency components
5. **With enough shots (1024), dominant peaks in QFT should align with FFT peaks** — but QFT won't show exact magnitude, only relative probabilities

The key insight: FFT gives you the exact spectrum. QFT gives you a probabilistic sampling of the spectrum. They agree on *which* frequencies dominate, but differ in *how* the result is represented.
