## Visual Design

Based on user's example:
- Blue-tinted background (`bg-blue-50/50`)
- Header with Binary icon + "Quantum Oracle Topography" title
- 4 cards in responsive grid (1→2→4 columns)
- Each card shows: case_id, n_qubits, Compass/Scale icon, truth table grid, classification
- State 0: white cell with blue border
- State 1: solid blue cell with white text
- Mount animation: translate-y + opacity transitions with staggered delays
