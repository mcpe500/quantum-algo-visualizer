## JSON Requirement Check

Requirement "harus pakai JSON" fulfilled.

Data flow:

`QFTSignalTopology` → `qftApi.getCases()` → `GET /api/qft/cases` → `get_qft_cases()` → `list_cases('qft', 'QFT-')` → load `datasets/qft/*.json`.

No hardcoded dataset array is used as primary source.
