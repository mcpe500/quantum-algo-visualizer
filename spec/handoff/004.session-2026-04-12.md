## Summary

Implemented quantum trace table for DJ with 5 steps and Dirac notation. Following user example format.

## What Was Done

### 1. Fixed Backend Trace

- Changed from 6 stages to 5 stages
- Stages: init → H+X → H oracle → Uf → measurement
- Added Dirac notation: |0⟩, |1⟩, |+⟩, |−⟩
- Keterangan column empty

### 2. New Files

- `backend/api/dj.py` - trace endpoint (lines 362-453)

### 3. Integration

## Format Verified

```
Step  Operasi                          q0    q1    q2    ancilla  Keterangan
1     Inisialisasi awal                 |0⟩   |0⟩   |0⟩   |0⟩      -
2     H pada q0..q2 dan X pada ancilla   |+⟩   |+⟩   |+⟩   |1⟩      -
3     H pada ancilla                    |+⟩   |+⟩   |+⟩   |−⟩      -
4     Oracle balanced, Uf               |+⟩   |+⟩   |+⟩   |−⟩      -
5     Measurement pada q0..q2            |0⟩   |0⟩   |1⟩   -        -
```
