Qiskit convention used throughout (rightmost char = qubit 0):
- `SparsePauliOp.from_list` тАФ native Qiskit convention
- `Statevector.expectation_value` тАФ consistent with circuit ordering
- ZZ Pauli string: `chars[n-1-qi] = 'Z'`, `chars[n-1-qj] = 'Z'`
- Bitstring decode: `bits[i] = int(bitstring[n-1-i])`

## Architecture Notes

### VQE Algorithm
```
Ansatz types:
  ry_linear:   Ry(╬╕с╡в) per qubit + linear CNOT chain (0тЖТ1тЖТ2тЖТ...тЖТn-1)
  ry_circular: Ry(╬╕с╡в) per qubit + circular CNOT ring (0тЖТ1тЖТ...тЖТn-1тЖТ0)

Energy evaluation per iteration:
  sv = Statevector(qc.assign_parameters(╬╕))
  E = Re(<sv|H_op|sv>)  тЖР expectation_value method

Optimizer: scipy COBYLA, maxiter=150, rhobeg=0.5, seed=42
```

### QAOA Algorithm
```
Cost Hamiltonian: H_C = ╬г_{(i,j)тИИE} (I - Z_iZ_j) / 2
Circuit: H^тКЧn тЖТ [cost(╬│) тЖТ mixer(╬▓)]^p тЖТ measure
  cost(╬│): for each edge (i,j): CNOT(i,j) тЖТ RZ(2╬│, j) тЖТ CNOT(i,j)
  mixer(╬▓): RX(2╬▓, i) for each qubit i

Expected cut: sv.expectation_value(H_C)
Optimizer: scipy COBYLA, maxiter=100, minimize(-expected_cut), seed=42
Initial params: uniform [0, ╧А/2]
Final measurement: AerSimulator, shots=1024
```
