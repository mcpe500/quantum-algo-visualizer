# SPEC 038: Route Completion and Deep Links

## Status
Implemented.

## Scope
This specification ensures feature inventory routes resolve directly instead of requiring manual tab selection after navigation.

## Implementation
- `/dj/animation`, `/qft/animation`, and `/qaoa/animation` open the corresponding algorithm page with the animation tab active.
- `/playground/circuit` opens Qubit Playground with Circuit Lab active.
- `/formulas/studio` and `/formulas/stories` open Formula Studio with the requested tab active.
- Core route table remains centralized in `frontend/src/App.tsx`.

## Verification
Static route audit checks all new deep links in the router table.
