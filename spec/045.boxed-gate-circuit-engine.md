# SPEC-045: Gate-Centric Boxed Circuit Engine

## Prompt
Ubah engine visualisasi circuit kuantum agar area circuit utama mengikuti aturan tegas:
1. Setiap gate wajib box jelas di atas wire.
2. Jangan pakai sphere/orb/badge/marker dekoratif sebagai bentuk utama gate.
3. Satu operasi gate = satu box solid, center-aligned ke wire dan kolom step.
4. Ukuran box konsisten antargate.
5. Label gate di dalam box (H, X, Y, Z, M, SWAP, CPHASE, dll).
6. Controlled gate harus tetap punya koneksi antarqubit, tetapi qubit terlibat tetap punya box.
7. Multi-qubit gate harus grouped boxes + koneksi vertikal.
8. Circuit column harus rapi dalam grid langkah.
9. Fokus visual gate-centric boxed design.
10. Visual state qubit boleh terpisah, tetapi area circuit utama wajib explicit boxed gates.

## Tujuan
- Menstandarkan visual gate lintas DJ, QFT, QAOA ke format box.
- Menghilangkan marker-only controlled gate.
- Menjaga readability seperti diagram textbook/editor circuit.

## Implementasi

### 1) Shared Gate Primitive Dipaksa Menjadi Box
Komponen gate shared yang sebelumnya berbasis sphere/disc/dot diubah menjadi box geometry:
- `LabeledSphereGate` -> box
- `LabeledDiscGate` -> box
- `ControlDot` -> box berlabel `C`
- `TargetMarker` -> box berlabel `X`

Detail:
- Semua memakai `boxGeometry`.
- Label selalu berada di dalam box.
- Opacity aktif/inaktif disamakan dengan style gate box yang sudah ada.

### 2) DJ Controlled Marker Bukan Dot Lagi
Di `scene-column.tsx`, marker oracle:
- `笳汁 dirender dengan `ControlDot` (sekarang box)
- `竓描 dirender dengan `TargetMarker` (sekarang box)
- Koneksi vertikal kontrol-target tetap dipertahankan.

Hasil: controlled operation tidak lagi garis + titik saja.

### 3) QFT Scene Menjadi Boxed per Kolom
Di `qft/animation/scene-primitives.tsx`:
- Import gate diubah ke `LabeledBoxGate`.
- Ditambahkan `gateSize` berbasis gap kolom agar ukuran konsisten.
- Phase render:
  - `init` -> box `ENC` pada seluruh wire
  - `hadamard` -> `HadamardGate` box
  - `phase_cascade` -> box `C` (control) + box `CP` (target) + koneksi vertikal
  - `swap` -> dua box `SWAP` + koneksi vertikal
  - `measurement` -> box `M` pada seluruh wire

Hasil: setiap langkah QFT punya representasi box eksplisit.

### 4) QAOA Circuit Gate Menjadi Box
Di `qaoa/animation/scene-primitives.tsx`:
- `LabeledDiscGate` diganti `LabeledBoxGate`.
- Superposition `H` diberi ukuran konsisten.
- Cost edge gate dirender sebagai box `C` dan `ZZ` + koneksi vertikal.
- Mixer dirender box `RX`.
- Measurement/update dirender box `M`.

Catatan: sphere yang tersisa pada QAOA scene hanya untuk node graph input/output (bukan gate circuit), sehingga tidak melanggar aturan gate-centric pada area circuit.
