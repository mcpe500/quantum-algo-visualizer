# SPEC 059: Formula Studio Undo/Redo and Project Import/Export

## Status
Implemented.

## Scope
This specification closes Formula Studio session durability gaps by adding reversible canvas operations and portable project files.

## Implementation
- `historyReducer.ts` adds a 50-entry bounded undo/redo stack for node add/remove/move, connection add/remove/update, property updates, clear, and load operations.
- `projectIO.ts` exports `.qav-project` JSON files with metadata and imports sanitized canvas state.
- `CanvasToolbar.tsx` exposes Undo, Redo, Export, and Import controls.
- `StudioCanvas.tsx` wires Ctrl+Z, Ctrl+Y, project import, project export, and history reset.

## Verification
Static TypeScript subset checking reaches only dependency-resolution errors caused by absent `node_modules`; no syntax errors are reported for the edited symbolic and project files.
