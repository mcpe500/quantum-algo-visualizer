# SPEC 048: QFT Classic Visualization Enhancement

## Prompt

Tambahkan visualisasi alur data (flow diagram) dan penjelasan lengkap untuk QFT Classic tab. User membutuhkan visualisasi yang jelas untuk buku TA, menunjukkan:
1. Alur transformasi dari Input Signal → Zero-Padding → FFT → Dominant Bins
2. Penjelasan detail tentang "Dominant Frequency Bins" - apa itu, kenapa dominant, dan kenapa ada 2 bin yang sama (conjugate symmetry)
3. Highlight visual untuk bin dominan di chart spektrum
4. Panah penghubung antara chart Input Signal dan FFT Spectrum

## Purpose

Membuat visualisasi QFT Classic lebih informatif dan educational untuk pembaca buku TA. Visualisasi harus menjelaskan:
- Bagaimana data mengalir dari input ke output
- Konsep frequency bin dan magnitude
- Fenomena conjugate symmetry pada FFT sinyal real
- Hubungan antara input signal dan frekuensi dominan

## Why

1. Visualisasi QFT Classic saat ini hanya menampilkan 2 chart berdampingan tanpa penjelasan hubungan antara keduanya
2. Pembaca buku TA perlu memahami konsep "Dominant Frequency Bins" yang merupakan hasil analisis FFT
3. Penjelasan tentang zero-padding preprocessing perlu divisualisasikan secara eksplisit
4. Konjugate symmetry (bin k dan bin N-k) sering membingungkan dan perlu dijelaskan
5. Flow diagram membantu pemahaman alur transformasi domain waktu → domain frekuensi

## Changes Planned

### 1. QFTFlowDiagram Component

Komponen baru yang menampilkan 4 step transformasi:
- **Sinyal Input**: Domain waktu, N titik data
- **Zero-Padding**: Penambahan nol untuk mencapai 2^n titik
- **FFT Algorithm**: Transformasi ke domain frekuensi
- **Frekuensi Dominan**: Hasil analisis - bin dengan magnitude tertinggi

Setiap step dihubungkan dengan panah dan label proses.

### 2. DominantBinsExplanation Component

Komponen interaktif dengan 3 section expandable:

**Section 1: "Apa itu Frequency Bin?"**
- Penjelasan frequency bin sebagai slot frekuensi
- Analogi: kotak-kotak yang menyimpan kekuatan sinyal per frekuensi
- Hubungan antara jumlah data dan jumlah bin

**Section 2: "Kenapa 'Dominant'?"**
- Penjelasan magnitude sebagai ukuran kekuatan frekuensi
- Analogi: nada paling keras dalam lagu
- Contoh perbandingan magnitude

**Section 3: "Kenapa Ada 2 Bin Sama?"**
- Penjelasan Conjugate Symmetry untuk sinyal real
- Rumus: Bin k ↔ Bin (N-k)
- Mengapa cukup melihat setengah pertama spektrum

**Plus:** Analisis hasil aktual dengan kalkulasi frekuensi (f = bin/N)

### 3. SpectrumChart Enhancement

Menambahkan:
- Prop `highlightBins` untuk menerima array bin yang perlu di-highlight
- Warna amber/orange untuk bar dominant bins
- Label angka di atas bar dominant
- Legend untuk membedakan dominant vs other bins

### 4. Visual Connector Between Charts

Menambahkan panah besar di tengah (desktop only) antara:
- Input Signal Chart (kiri)
- FFT Spectrum Chart (kanan)

Menggunakan ikon ArrowRight dengan background bulat.

### 5. Integration in QFTClassicTab

Mengupdate QFTClassicTab untuk:
- Render QFTFlowDiagram di atas metrics
- Tambahkan DominantBinsExplanation di bawah section Dominant Frequency Bins
- Highlight dominant bins di SpectrumChart
- Tambahkan panah connector antar chart

## Logic / Pseudocode

```text
QFTFlowDiagram:
  Input: nPointsOriginal, nPointsPadded, dominantBins
  
  Step 1: Sinyal Input
    - Icon: Signal
    - Label: "{nPointsOriginal} titik data"
    - Domain: Waktu
  
  Arrow 1: "Zero-Padding"
  
  Step 2: Sinyal Pad
    - Icon: FunctionSquare
    - Label: "{nPointsPadded} titik (2^n)"
    - Note: Persiapan FFT
  
  Arrow 2: "FFT Algorithm"
  
  Step 3: Spektrum FFT
    - Icon: BarChart3
    - Label: "{nPointsPadded} bins"
    - Domain: Frekuensi
  
  Arrow 3: "Analisis"
  
  Step 4: Frekuensi Dominan
    - Icon: Sparkles
    - Label: dominant bins
    - Hasil analisis

DominantBinsExplanation:
  Input: dominantBins, dominantMagnitudes, nPoints
  
  Section 1 (Expandable): Apa itu Frequency Bin?
    - Penjelasan konsep bin
    - Analogi slot frekuensi
    - Hubungan N data → N bin
  
  Section 2 (Expandable): Kenapa "Dominant"?
    - Penjelasan magnitude
    - Analogi nada paling keras
    - Contoh nilai magnitude
  
  Section 3 (Expandable): Kenapa 2 Bin Sama?
    - Penjelasan Conjugate Symmetry
    - Rumus: k ↔ (N-k)
    - Note: cukup lihat setengah pertama
  
  Analysis Panel:
    - Grid bin cards dengan frekuensi
    - Mark mirror pairs
    - Formula: f = bin/N

SpectrumChart:
  Input: data, highlightBins
  
  For each bar:
    If bin in highlightBins:
      Fill: amber (#f59e0b)
      Opacity: 1
      Add label on top
    Else:
      Fill: blue (#2563EB)
      Opacity: 0.8
  
  Add legend if highlightBins provided
```

## Design Notes

- Flow diagram menggunakan gradient background untuk membedakan dari content utama
- Warna step: Input (white) → Pad (blue-50) → Spectrum (indigo-50) → Dominant (amber-50)
- Icons dari Lucide React untuk konsistensi
- Dominant bins menggunakan warna amber/orange untuk menarik perhatian
- Explanation sections menggunakan accordion pattern dengan ikon toggle
- Semua perubahan tetap mengikuti existing design system (Tailwind CSS)
