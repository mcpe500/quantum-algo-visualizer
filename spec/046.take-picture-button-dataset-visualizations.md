# SPEC 046: Take Picture Button for Dataset Visualizations

## 壹 Task
Add per-card "Take Picture" button to dataset visualization pages (DJ Oracle Topography, QFT Signal Topology, QAOA Dataset Visualizer) for thesis book (BUKU TA) screenshots. Only screenshot - no video.

## 貳 Goals
- Enable per-card screenshot capture of dataset visualization cards
- Use existing `downloadElementAsPNG` infrastructure (html2canvas)
- Per-card button placement (user confirmed)
- Camera icon + "Take Picture" text

## 叁 Why
User needs high-quality PNG images of dataset visualizations for inclusion in thesis book. Combined algorithm pages have Download PNG, but dataset topography pages lacked this feature.

## 肆 Pages Modified

| Page | File | Capture ID Pattern |
|------|------|-------------------|
| DJ Topography | `OracleTopography.tsx` | `dj-oracle-{case_id}` |
| QFT Topography | `QFTSignalTopology.tsx` | `qft-signal-{case_id}` |
| QAOA Dataset | `QAOADatasetVisualizer.tsx` | `qaoa-dataset-{case_id}` |

## 伍 Implementation Details

### Behavior (Updated)
1. User clicks button → button immediately hidden (replaced by nothing for 10 seconds)
2. After 50ms delay → html2canvas captures (button already hidden)
3. Download completes
4. Wait 10 seconds from click
5. Button reappears

**Key:** Both button AND "Capturing..." text are hidden during the 10-second cooldown. No text or icon appears in the captured PNG.

1. **Imports Added:**
   - `Camera` from `lucide-react`
   - `downloadElementAsPNG` from `../../utils/download`

2. **DatasetCard Changes:**
   - Add `useState` for `isCapturing` local state
   - Add `handleTakePicture` async function
   - Add `id={algo-{case_id}}` to card wrapper div
   - Add button with Camera icon in header section

### OracleTopography.tsx Changes (lines 42-93)
```tsx
const [isCapturing, setIsCapturing] = useState(false);

const handleTakePicture = () => {
  if (isCapturing) return;
  setIsCapturing(true);
  setTimeout(() => {
    downloadElementAsPNG(`dj-oracle-${data.case_id}`, `dj-oracle-${data.case_id}.png`)
      .finally(() => {
        setTimeout(() => setIsCapturing(false), 10000);
      });
  }, 50);
};

<div id={`dj-oracle-${data.case_id}`} ...>

{!isCapturing && (
  <button
    onClick={handleTakePicture}
    className="p-2 rounded-lg border border-gray-200 text-gray-500 hover:bg-gray-50 transition-all"
  >
    <Camera size={18} />
  </button>
)}
```
