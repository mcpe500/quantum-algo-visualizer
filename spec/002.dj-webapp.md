## 陸　Logic & Pseudocode

### Classical Algorithm (Brute Force)

```python
function brute_force(truth_table, n_qubits):
    first_output = truth_table[F(0)]
    for i in range(1, 2^(n-1) + 1):
        output = truth_table[F(i)]
        if output != first_output:
            return "BALANCED"
    return "CONSTANT"
```

Time Complexity: O(2^(n-1) + 1) worst case

### Quantum Algorithm

```python
function quantum_dj(truth_table, n_qubits, shots=1024):
    # 1. Initialize: |0>^n |0>
    # 2. Apply X then H to ancilla: |0>^n |->
    # 3. Apply H to input qubits: superposition
    # 4. Apply oracle U_f (once)
    # 5. Apply H to input qubits: interference
    # 6. Measure
    circuit = build_dj_circuit(truth_table, n_qubits)
    result = run_on_aer_simulator(circuit, shots)
    if counts == {'0'*n: shots}:
        return "CONSTANT"
    else:
        return "BALANCED"
```

Time Complexity: O(1) — one oracle evaluation
