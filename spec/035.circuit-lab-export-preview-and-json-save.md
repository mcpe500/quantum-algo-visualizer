# SPEC 035: Circuit Lab Export Preview and JSON Save

## Prompt

> "qubit nya sekalian buatkan supaya bisa di + ato - kan saja"
> "terus datanya bisa disimpan as json"
> "terus bisa export as projectq syntax, sama bisa export as cirq syntax, sama bisa export as qiskit syntax"
> "karena export ini berarti bisa keliatan outputnya"
> "jadi di SHOW kan kalau kode nya cirq/projectq/qiskit bakalan gimana"

## Tujuan

Menambahkan tiga kemampuan baru pada `Circuit Lab`:

- kontrol jumlah qubit memakai tombol `+/-`
- project saat ini bisa disimpan sebagai file JSON
- sintaks ekuivalen `Qiskit`, `Cirq`, dan `ProjectQ` ditampilkan langsung di UI dan bisa diunduh

## Kenapa

Circuit Lab tidak cukup hanya interaktif di dalam playground. User ingin melihat bagaimana sirkuit yang disusun akan terlihat bila diterjemahkan ke framework kuantum yang umum dipakai. Karena itu ekspor tidak boleh tersembunyi sebagai aksi download saja; hasilnya harus tampil langsung agar bisa dibaca, dicek, dan dibandingkan.

## Scope

- Tetap di tab `Circuit Lab`
- Tidak menambah dependency baru
- Menjaga engine simulasi tetap sama
- Menjaga batas qubit saat ini tetap `1..3`
- Menambah preview ekspor untuk:
  - `JSON`
  - `Qiskit`
  - `Cirq`
  - `ProjectQ`

## Perubahan Utama

### 1. Kontrol qubit diubah ke plus/minus

Toolbar Circuit Lab sebelumnya memakai tombol tetap `1/2/3`. Sekarang diganti ke pola yang sama dengan kontrol kolom:

- tombol `-`
- angka tengah `n qubits`
- tombol `+`

Hook builder menambahkan:

- `canIncreaseQubits`
- `canDecreaseQubits`
- `increaseQubits()`
- `decreaseQubits()`

### 2. Project data bisa diserialisasi ke JSON

Hook `useCircuitBuilder` menambahkan model serialisasi baru:

```ts
interface CircuitProjectData {
  version: 1;
  numQubits: number;
  columnCount: number;
  placements: CircuitPlacement[];
}
```

Lalu mengekspos:

```ts
exportProjectData(): CircuitProjectData
```

JSON ini merepresentasikan project builder secara langsung, tanpa menyimpan state turunan seperti preview, trace, atau insight.

### 3. Hook menambahkan exporter syntax

Hook juga menambahkan:

```ts
type CircuitCodeExportFormat = 'json' | 'qiskit' | 'cirq' | 'projectq';
exportCircuitCode(format: CircuitCodeExportFormat): string
```

Exporter membaca placement yang sudah diurutkan berdasarkan `column`, lalu `row`, sama seperti replay internal.

### 4. Mapping syntax per framework

#### Qiskit

- `QuantumCircuit(n)`
- `qc.h`, `qc.x`, `qc.y`, `qc.z`, `qc.s`, `qc.t`
- `qc.rx`, `qc.ry`, `qc.rz`
- `qc.cx`, `qc.swap`, `qc.cp`

#### Cirq

- `cirq.LineQubit.range(n)`
- `cirq.H`, `cirq.X`, `cirq.Y`, `cirq.Z`, `cirq.S`, `cirq.T`
- `cirq.rx(theta).on(...)`, `cirq.ry(theta).on(...)`, `cirq.rz(theta).on(...)`
- `cirq.CNOT`, `cirq.SWAP`, `cirq.CZPowGate(exponent=theta/np.pi)`

#### ProjectQ

- `MainEngine()`
- `H | qubits[i]`, `X | qubits[i]`, dan seterusnya
- `Rx(theta) | qubits[i]`, `Ry(theta) | qubits[i]`, `Rz(theta) | qubits[i]`
- `CNOT | (...)`
- `Swap | (...)`
- `with Control(eng, qubits[c]): R(theta) | qubits[t]` untuk controlled phase

### 5. Export Preview ditampilkan langsung di UI

Ditambahkan panel baru `Export Preview` di bawah workspace utama. Panel ini berisi:

- tab `JSON`, `Qiskit`, `Cirq`, `ProjectQ`
- preview kode dalam blok monospaced
- tombol unduh untuk format aktif

Dengan ini user bisa langsung melihat hasil translasi sirkuit tanpa harus mengunduh dulu.

### 6. Save JSON tetap tersedia sebagai aksi cepat

Toolbar atas tetap diberi tombol `Save JSON` agar project bisa langsung disimpan sebagai file `.json` kapan pun dibutuhkan.

## Manual Testing Plan

1. Buka `/playground`
2. Masuk ke tab `Circuit Lab`
3. Ubah jumlah qubit memakai tombol `+/-`
4. Pastikan angka qubit berubah dan tetap dibatasi `1..3`
5. Tambahkan beberapa gate ke grid
6. Buka panel `Export Preview`
7. Pindah tab `JSON`, `Qiskit`, `Cirq`, `ProjectQ`
8. Pastikan isi kode berubah sesuai format aktif
9. Klik `Save JSON`
10. Pastikan file `.json` terunduh
11. Klik `Download Qiskit` atau download dari format aktif
12. Pastikan file `.py` terunduh dengan isi sesuai panel preview
13. Ubah angle gate parametrik
14. Pastikan preview export ikut berubah

## Catatan

1. Session ini hanya menambah ekspor satu arah dan preview kode. Import kembali dari JSON belum ditambahkan.
2. JSON menyimpan data project inti, bukan snapshot turunan seperti preview atau insight.
3. Output kode bersifat scaffold edukatif yang setara dengan circuit builder saat ini, bukan pipeline eksekusi penuh.
