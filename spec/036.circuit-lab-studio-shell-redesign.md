# SPEC 036: Circuit Lab Studio Shell Redesign

## Prompt

> "tolong buatkan seperti ini"
> "ini referensi"
> "continue"

## Tujuan

Mengubah `Circuit Lab` dari susunan kartu biasa menjadi studio shell yang jauh lebih dekat ke referensi: sidebar kiri untuk gate, stage tengah bertitik, inspector kanan bertab, dan timeline footer di bagian bawah.

## Latar Belakang

Implementasi sebelumnya sudah fungsional, tetapi belum terasa seperti workspace studio. Struktur visual masih campur antara panel biasa dan area kerja. Referensi yang diberikan menuntut hierarki yang lebih tegas:

- top control bar
- left tool sidebar
- central stage
- right inspector with tabs
- bottom playback dock

Karena itu perubahan dilakukan pada level shell UI, bukan pada engine simulasi.

## Scope

- Tetap di route `/playground`
- Tetap pada tab `Circuit Lab`
- Tidak menambah dependency baru
- Tidak mengubah engine `useCircuitBuilder`
- Menjaga fitur yang sudah ada:
  - drag and drop gate
  - playback per langkah
  - selected gate inspector
  - save JSON
  - preview source code Qiskit/Cirq/ProjectQ

## Perubahan Utama

### 1. Circuit Lab di-rewrite menjadi studio shell

`CircuitLabTab.tsx` dibangun ulang menjadi struktur penuh:

- top bar internal `StudioLab`
- left sidebar gate palette
- center dotted stage
- right inspector bertab `Visualize / Source Code`
- bottom timeline footer

### 2. Gate palette dipindah menjadi sidebar tool permanen

Sidebar kiri sekarang berisi:

- section title `Gerbang Quantum`
- grouped gate assets
  - `1-Qubit Basic`
  - `Parametrik`
  - `Multi-Qubit`
- status card di bagian bawah

Setiap gate tetap draggable, tetapi visualnya sekarang mengikuti pola asset card seperti referensi.

### 3. Stage tengah memakai dotted workspace

Area tengah sekarang menggunakan background bertitik dengan card sirkuit yang diletakkan di tengah workspace. Grid sirkuit tetap mempertahankan logika builder lama, termasuk:

- drop target per sel
- highlight kolom aktif
- validasi gate dua qubit
- connector control-target
- drag untuk memindahkan placement

### 4. Inspector kanan dipecah menjadi dua tab

#### Visualize

Berisi:

- monitor Bloch sphere per qubit
- probability distribution
- automated insights
- gate inspector untuk edit/remove gate terpilih

#### Source Code

Berisi:

- chip exporter `Qiskit`, `Cirq`, `ProjectQ`
- code preview dalam panel gelap
- tombol copy code di top bar tetap bekerja terhadap format aktif

JSON tidak ditaruh sebagai tab code karena peran utamanya adalah save/download project.

### 5. Footer timeline dijadikan dock playback

Playback control sekarang diposisikan di footer studio dengan pola:

- previous / play-pause / next
- slider timeline di tengah
- state vector dan ringkasan step di kanan

Ini membuat alur interaksi lebih dekat ke referensi visual yang diberikan.

### 6. Wrapper halaman dibuat lebih immersive saat tab circuit aktif

`QubitPlaygroundPage.tsx` disesuaikan agar mode `Circuit Lab` tidak lagi memakai header halaman sebesar mode `State Lab`.

Perubahan:

- `pageWidthClass` untuk circuit diubah menjadi `max-w-none`
- `Circuit Lab` memakai header halaman yang lebih ringkas
- tab switch tetap tersedia, tetapi studio shell menjadi elemen dominan di layar

### 7. Responsivitas studio dimajukan ke `lg`

Sebelumnya layout tiga panel hanya aktif di `xl`. Kini shell diatur mulai `lg` sehingga pada desktop menengah tetap terasa seperti studio, bukan stack vertikal panjang.

## Keputusan Desain

1. `useCircuitBuilder` dipertahankan apa adanya sebagai engine domain.
2. `CircuitLabTab` menjadi lapisan presentasional baru.
3. Inspector kanan tidak lagi memakai panel terpisah di bawah halaman; semuanya dipusatkan ke tab kanan.
4. JSON save tetap tersedia sebagai aksi global, tetapi code tab difokuskan ke framework syntax.
5. Final `StateInfoPanel` untuk mode circuit dihapus dari shell utama agar layout lebih dekat ke referensi studio.

## Catatan

1. Redesign ini fokus pada layout shell dan hierarki visual, bukan perubahan engine simulasi.
2. Shell studio masih menjaga fallback responsif untuk layar kecil, tetapi pola tiga panel kini aktif mulai `lg`.
3. Jika masih ingin lebih dekat ke referensi, langkah berikutnya yang paling natural adalah menambah quick gate search dan compact mobile drawer mode.
