## Logic / Pseudocode

### VQE

```
load case → extract n_qubits, ansatz_type, n_layers, hamiltonian_terms

classical FCI:
  H_op = SparsePauliOp.from_list(hamiltonian_terms)
  H_matrix = H_op.to_matrix()
  eigenvalues = np.linalg.eigvalsh(H_matrix)
  fci_energy = eigenvalues[0]  # minimum = ground state

quantum VQE:
  build ansatz circuit with ParameterVector θ
  energy_fn(params):
    bind params → qc_bound
    sv = Statevector(qc_bound)
    return Re(<sv|H_op|sv>)
  x0 = random uniform [0, π] shape (n_qubits * n_layers,)
  result = scipy COBYLA minimize(energy_fn, x0, maxiter=150)
  convergence_history tracked per energy_fn call

report:
  energy_error = |vqe_energy - fci_energy|
  accuracy = (1 - energy_error / |fci_energy|) * 100
```

### QAOA

```
load case → extract n_nodes, edges, p_layers

exact:
  brute-force 2^n partitions → max cut

simulated annealing:
  random start, T=2.0, decay=0.99, 500 iters, seed=42
  flip random bit, accept if better or with P=exp(Δ/T)

quantum QAOA:
  H_C = Σ_{(i,j)∈E} (I - Z_iZ_j) / 2  (SparsePauliOp)
  
  qaoa_circuit_no_measure(gamma, beta):
    H on all qubits
    for each layer l:
      for each edge (i,j): CNOT(i,j) → RZ(2γ_l, j) → CNOT(i,j)
      for each qubit i: RX(2β_l, i)
  
  expected_cut(params):
    sv = Statevector(qaoa_circuit(gamma, beta))
    return Re(<sv|H_C|sv>)  # expectation = expected cut value
  
  x0 = random uniform [0, π/2] shape (2*p_layers,)
  result = COBYLA minimize(-expected_cut, x0, maxiter=100)
  
  final: run circuit with AerSimulator(shots=1024)
  top bitstrings → cut_distribution, best_cut, expected_cut

report:
  approx_ratio = best_cut / exact_cut
  three-way comparison: exact vs SA vs QAOA
```

## Qubit Ordering Convention

Qiskit convention used throughout: **rightmost character = qubit 0**.

- `SparsePauliOp.from_list` uses Qiskit convention natively.
- `Statevector(qc).expectation_value(H_op)` is consistent with circuit ordering.
- `build_zz_pauli_string(n, qi, qj)`: `chars[n-1-qi] = 'Z'`, `chars[n-1-qj] = 'Z'`.
- Bitstring decoding: `bits[i] = int(bitstring[n-1-i])`.
