# SPEC 032: Qubit Playground

## Prompt

> "kalo saya mau nambah semacam extra tools jadi ada fitur untuk visualisasi qubit nya, jadi bisa nunjukkan fase fase nya."
> "bukan, ini murni untuk main main sama qubit nya doang. belum ada algoritma nya dulu apa apa. cuman threejs yang bisa muter muter bisa ada qubitnya. terus bisa ada beberapa skenario qubit. kalo kasus |0>, |1> dan lain lain lengkap tapi."

Tambahan klarifikasi:

- Preset states + Gate buttons (H, X, Y, Z, S, T)
- Hapus semua shared phase files (PhaseWheel, PhaseEvolutionChart, QubitPhasePanel, phase-utils)

## Goal

Halaman interaktif standalone untuk "main-main" dengan qubit — murni frontend, tanpa backend. User bisa pilih preset state, apply gate, lihat state berubah di 3D Bloch sphere, dan baca informasi state (θ, φ, P(0), P(1)).

## Why

Fondamental sebelum masuk algoritma DJ/QFT/QAOA. User ingin playground qubit yang bisa di-interaksi — pilih state, apply gate, lihat hasil. Ini bukan phase explorer yang pakai data dari backend animation endpoint.

## Route

`/playground` — standalone page, tidak ada hubungannya dengan algorithm pages.

## Math Details

### Preset States

| Label | Ket | θ | φ | Bloch (x,y,z) |
|-------|-----|---|---|---------------|
| \|0⟩ | \|0⟩ | 0 | 0 | (0, 0, 1) |
| \|1⟩ | \|1⟩ | π | 0 | (0, 0, -1) |
| \|+⟩ | (\|0⟩+\|1⟩)/√2 | π/2 | 0 | (1, 0, 0) |
| \|−⟩ | (\|0⟩−\|1⟩)/√2 | π/2 | π | (-1, 0, 0) |
| \|+i⟩ | (\|0⟩+i\|1⟩)/√2 | π/2 | π/2 | (0, 1, 0) |
| \|−i⟩ | (\|0⟩−i\|1⟩)/√2 | π/2 | 3π/2 | (0, -1, 0) |

### Gate Matrices

```
H = 1/√2 [[1,1],[1,-1]]
X = [[0,1],[1,0]]
Y = [[0,-i],[i,0]]
Z = [[1,0],[0,-1]]
S = [[1,0],[0,i]]
T = [[1,0],[0,e^(iπ/4)]]
```

### State Update

1. Apply gate matrix to (α, β) complex vector
2. Normalize result
3. Extract new θ, φ from normalized complex amplitudes
4. Compute P(0) = |α|², P(1) = |β|²
5. Record in history

## Test Cases

| Action | Expected |
|--------|----------|
| Load `/playground` | Bloch sphere renders, state = \|0⟩, info shows P(0)=100% |
| Click \|+⟩ preset | State vector animates to equator (+X), P(0)=50%, P(1)=50% |
| Click X gate on \|0⟩ | State becomes \|1⟩, vector at south pole |
| Click H then X on \|0⟩ | \|0⟩→H→\|+⟩→X→\|−⟩ |
| Click Undo | State returns to previous in history |
| Click Reset | State returns to \|0⟩, history cleared |
| Drag on canvas | OrbitControls rotates camera |
| TypeScript build | No errors |
