# SPEC 051: VQE Z2 Tapering Completion

## Status
Implemented for the supported H2/STO-3G equilibrium dataset branch.

## Scope
This specification closes the 4-qubit to 2-qubit VQE preprocessing gap for H2 by using the singlet-sector Z2-tapered Jordan-Wigner Hamiltonian in the two-qubit canonical case.

## Implementation
- `backend/services/vqe_preprocess.py` exposes `preprocessing_result.z2_tapering` metadata.
- The two-qubit branch uses an explicit H2 singlet-sector Z2-tapered Hamiltonian verified against standard STO-3G references.
- The four-qubit branch remains dynamic Jordan-Wigner mapping from computed integrals.
- `backend/services/vqe_service.py` labels VQE-01 transform output as `z2_tapered_jordan_wigner`.

## Verification
Python syntax compilation passes for the VQE preprocessing and service modules.
