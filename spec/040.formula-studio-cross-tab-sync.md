# SPEC 040: Formula Studio Cross-Tab Highlight Sync

## Status
Implemented.

## Scope
This specification connects Formula Studio Stories and Studio through shared state, enabling formula-focused navigation between narrative steps and editable graph nodes.

## Implementation
- `FormulaStudioContext.tsx` provides a Context API synchronization channel.
- Stories requests open and highlight a formula in Studio.
- Studio node selection publishes the active formula so Stories can jump to the related step.
- When a requested formula is not already on the canvas, Studio creates a formula node at the visible center and selects it.

## Verification
Static audit confirms the shared context and route-level Studio/Stories entries are present.
