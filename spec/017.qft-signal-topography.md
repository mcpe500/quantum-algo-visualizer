# SPEC 017: QFT Signal Topography (JSON-Driven)

## Prompt

Implement halaman visualisasi QFT dengan gaya kartu + waveform SVG seperti contoh user, lalu integrasikan ke `quantum-algo-visualizer`.

Constraint penting dari user: **harus pakai JSON dataset**, bukan hardcoded array di frontend.

## Tujuan

1. Menyediakan halaman visual ringkas untuk membaca pola sinyal QFT case.
2. Menjaga konsistensi visual dengan desain yang user berikan.
3. Menjamin sumber data berasal dari dataset JSON backend.

## Konteks Codebase

- Frontend: React + Vite + Tailwind + lucide-react.
- API client: `frontend/src/services/api.ts` (`qftApi.getCases()`).
- Backend route: `GET /api/qft/cases`.
- Backend service: `get_qft_cases()` → `list_cases('qft', 'QFT-')`.
- Loader hanya baca `.json` dari `datasets/qft`.

## Implementasi

### 1) Komponen baru: `QFTSignalTopology`

File: `frontend/src/components/qft/QFTSignalTopology.tsx`

- Dataset source: `qftApi.getCases()` (JSON dari backend).
- State: `cases`, `loading`, `error`, `mounted`.
- Sorting case_id numeric (`QFT-01`, `QFT-02`, ...).
- Kartu dataset:
  - Header: case ID, signal type, icon contextual (`Waves`/`Activity`).
  - Body: native SVG waveform dengan grid, zero-line, gradient area, hover points.
  - Footer: deskripsi + badge jumlah point.
- UX states:
  - loading spinner,
  - error card + retry,
  - empty dataset message.

### 2) Routing page baru

File: `frontend/src/pages/QFTTopographyPage.tsx`

- Wrapper page dengan back link ke selector.
- Render `<QFTSignalTopology />`.

### 3) Route registration

File: `frontend/src/App.tsx`

- Tambah route: `/qft/topography`.

### 4) Navigasi dari halaman benchmark QFT

File: `frontend/src/pages/QFTCombinedPage.tsx`

- Tambah link `Signal Topography` menuju `/qft/topography`.

### 5) Barrel export

File: `frontend/src/components/qft/index.ts`

- Export `QFTSignalTopology`.

## Validasi JSON Source

Alur data final:

`QFTSignalTopology` → `qftApi.getCases()` → `/api/qft/cases` → `get_qft_cases()` → `list_cases('qft','QFT-')` → baca file `.json`.

Tidak ada array dataset hardcoded dipakai sebagai source utama.
